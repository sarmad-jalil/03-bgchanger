# Background Changer

A simple React application that allows users to change the background color of the page by clicking on color buttons.

## Features

- Click on colored buttons to change the page background
- Smooth color transition with 200ms duration
- Available colors: Red, Green, Blue, Purple, Yellow, Orange

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
